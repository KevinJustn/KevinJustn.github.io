PokémonRPG - Swift and Xcode
This project was one of the final assignments in my university course for iOS Development. The project was based on previous assignments we had completed, but expanded on it with new features or requirements that made it more difficult to complete. Coding all the files and making sure testing went as planned took a majority of the time, especially with testing the new features and researching the best way to implement them.
The final result is a mock Pokémon game where the user controls their character sprite with an on-screen controller and can "collect" Pokémon by running into them. When a Pokémon is captured, music is played to notify the player. There is also background music playing in the app. The user can then see how many of the original 151 Pokémon they captured by clicking the Pokéball icon.

Transfer Course Sheet - JavaScript with React and Bootstrap
This project was used as a final assignment in my User Interfaces course to test our knowledge and capabilites in Javascript and its frameworks. The project was an extension on what we had previously worked on in the course, but with more specifications and event changes. This project was a major stepping stone in developing my website as it provided crucial functionalities that I would use.
The final result is a mock Transfer Student Course Sheet in which the user can input their name and total amount of transfer credits they have. Depending on the amount of transfer credits they enter, the amount of courses listed in Speaking and Writing Emphasis will change. In addition, there is an editing function in which users can change and input data into the tables to more accurately detail the classes.
